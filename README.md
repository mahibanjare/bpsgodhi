# Bright Public School Website

**CBSE English Medium School | Naya Raipur, Chhattisgarh**

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
